How to run SmartPen :
 * go in the server/ directory
 * run SmartPen.jar (java -jar SmartPen.jar)
 * enter the server IP address in app/src/main/java/apps/Login.java serverIP field
	for exemple for a server running at 192.168.1.4, edit the line :
            public static final byte[] serverIP = {(byte)10,(byte)0,(byte)1,(byte)4};
	to this :
	    public static final byte[] serverIP = {(byte)192,(byte)168,(byte)1,(byte)4};
 * install SmartPen on the smartphone
